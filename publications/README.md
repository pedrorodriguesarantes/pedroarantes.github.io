# Self-hosted paper PDFs

Drop author-accepted manuscripts here, named after the DBLP key with
slashes replaced by dashes, e.g.:

    conf/icse/SteinmacherCTG16  ->  conf-icse-SteinmacherCTG16.pdf

`missing_preprints.md` (generated by `scripts/build_pubs.py`) tells you the
exact expected filename for every paper that still needs one. Papers found
here get a [PDF] badge on the publications page — used only when the paper
is neither open access nor on arXiv.
